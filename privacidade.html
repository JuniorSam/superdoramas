Política de Privacidade

Última atualização: 28 de abril de 2026  ·  Aplicável ao app Super Doramas (Android)

O Super Doramas respeita sua privacidade. Esta política explica de forma clara quais dados coletamos, como os usamos e quais são seus direitos. Leia com atenção antes de usar o aplicativo.

1. Quem somos

O Super Doramas é um aplicativo de organização e tracker de K-Dramas e C-Dramas desenvolvido por SD Entretenimento, com sede em Itabira, MG, Brasil. Para contato: juniorsam2011@gmail.com.

2. Dados que coletamos

Coletamos apenas os dados necessários para o funcionamento do app:

  • Dados de conta: nome de usuário, endereço de e-mail e senha (armazenados com criptografia no Firebase Authentication).
  • Dados de uso: dramas adicionados à biblioteca, episódios marcados, reviews escritas, XP e rank acumulados.
  • Dados de chat: mensagens enviadas no chat da comunidade (públicas para todos os usuários do app).
  • Dados de dispositivo: identificador de dispositivo e sistema operacional, coletados pelo Google AdMob para exibição de anúncios personalizados.
  • Dados de compra: status de assinatura VIP gerenciado pelo RevenueCat. Dados de pagamento são processados exclusivamente pelo Google Play — não temos acesso a eles.

3. Como usamos seus dados

  • Autenticar seu acesso e sincronizar sua biblioteca entre dispositivos.
  • Calcular XP, rank e exibir conquistas no seu perfil.
  • Exibir anúncios relevantes (usuários sem VIP) via Google AdMob.
  • Gerenciar sua assinatura VIP via RevenueCat.
  • Melhorar o app com base em dados agregados e anônimos de uso.

4. Compartilhamento de dados

Não vendemos nem compartilhamos seus dados pessoais com terceiros, exceto com os seguintes parceiros de serviço essenciais ao funcionamento do app:

  • Google Firebase — autenticação, banco de dados e configuração remota.
  • Google AdMob — exibição de anúncios para usuários gratuitos.
  • RevenueCat — gerenciamento de assinaturas VIP.
  • TMDB (The Movie Database) — catálogo de dramas (apenas consultas, nenhum dado seu é enviado).

5. Anúncios e personalização

O app exibe anúncios via Google AdMob para usuários sem assinatura VIP. O AdMob pode usar o identificador de publicidade do seu dispositivo (GAID) para personalizar anúncios. Você pode desativar a personalização nas configurações do seu dispositivo Android em Configurações → Google → Anúncios. Usuários VIP não visualizam anúncios.

6. Retenção de dados

Seus dados são mantidos enquanto sua conta estiver ativa. Ao excluir sua conta pelo app, todos os seus dados pessoais (perfil, biblioteca, reviews e mensagens de chat) são permanentemente removidos dos nossos servidores em até 30 dias.

7. Segurança

Utilizamos criptografia SSL/TLS para todas as comunicações entre o app e nossos servidores. Senhas são armazenadas com hash seguro pelo Firebase Authentication. Nenhum dado de pagamento transita por nossos sistemas.

8. Crianças

O Super Doramas não é destinado a crianças menores de 13 anos. Não coletamos intencionalmente dados de menores. Se identificarmos que coletamos dados de uma criança sem consentimento dos pais, removeremos esses dados imediatamente.

9. Seus direitos

Você tem direito a:

  • Acessar, corrigir ou excluir seus dados pessoais.
  • Revogar o consentimento para uso de dados a qualquer momento.
  • Solicitar a portabilidade dos seus dados.
  • Registrar reclamações junto à ANPD (Autoridade Nacional de Proteção de Dados).

Para exercer qualquer desses direitos, entre em contato: juniorsam2011@gmail.com.

10. Alterações nesta política

Podemos atualizar esta política periodicamente. Quando houver mudanças significativas, notificaremos via atualização do app ou e-mail. A data de "Última atualização" no topo deste documento sempre refletirá a versão mais recente.
